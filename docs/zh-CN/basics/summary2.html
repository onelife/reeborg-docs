15. 小结¶

是时候回顾一下，从上一次小结到现在为止所学的内容了。

其间，我们第一次见到了 Python 关键字：def 、 from 和 import ，还有怎样把函数存在库里备用。

我们学习了新概念——“循环”，和乐跑的世界所独有的关键字—— repeat 。

我们用到了五个乐跑的世界所专用的函数：

  • move()
  • turn_left()
  • take()
  • put()
  • pause()

现在，我们要开始学习一些非常重要的编程概念，涉及在不同的可能性之间自动地进行选择。
